11 """ 

12 Weighted Directional Derivative-Enhanced Gaussian Process (wdDEGP) model. 

13 

14 This model augments standard GP regression by incorporating weighted directional derivatives 

15 through a hypercomplex framework. It supports submodel decomposition to handle multiple 

16 directional structures and adapts to anisotropic kernels. 

17 

18 Parameters 

19 ---------- 

20 x_train : ndarray 

21 Training input locations (N, d). 

22 y_train : list of arrays 

23 List of training outputs, including function values and derivatives. 

24 n_order : int 

25 Order of directional derivatives. 

26 n_bases : int 

27 Number of hypercomplex basis components. 

28 index : list of lists 

29 Submodel point indices. 

30 der_indices : list of lists 

31 Multi-index derivative directions for each submodel. 

32 rays : list of ndarray 

33 Directional vectors associated with each submodel. 

34 normalize : bool, default=False 

35 Whether to normalize the data. 

36 sigma_data : ndarray or None 

37 Noise variances for each data point. 

38 kernel : str, default="SE" 

39 Kernel type ('SE', 'RQ', etc). 

40 kernel_type : str, default="anisotropic" 

41 Use 'anisotropic' or 'isotropic' kernel behavior. 

42 """
